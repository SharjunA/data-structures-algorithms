# Data Structures and Algorithms

This repository contains implementations of fundamental **Data Structures and Algorithms** in **Python** and **Java**.

The goal of this repository is to build a strong understanding of algorithmic concepts and maintain a structured reference for **coding interviews and technical assessments**.

---

## Topics Covered

Current topics in this repository include:

- Sorting Algorithms
- Object-Oriented Programming Concepts

More topics will be added over time such as:

- Linked Lists
- Stacks & Queues
- Trees
- Graphs
